# TODOMVC

Author: 1751358 李澎生

Preview：http://49.235.158.28/

## 功能
### 基本功能

- [x] 新增`Todo`
- [x] 删除`Todo`（拖拽删除）
- [x] 展现`Todo`列表
- [x] 全部完成/取消
- [x] 删除已完成
- [x] 保存页面状态，刷新页面后可恢复

### 高级功能

- [x] 过滤（`All`、`Ongoing`、`Completed`）
- [x] 编辑单条`Todo`
- [x] 添加`Todo`的`Deadline`
- [x] 以`Deadline`分块显示`TodoList`
- [x] 按照`Deadline`的剩余天数改变颜色
- [x] 拖拽单条`Todo`换位
- [x] 拖拽整个`Deadline`分块换位
- [x] 切换颜色主题
